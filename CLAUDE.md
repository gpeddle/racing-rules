## Project Overview

Mobile app to help users understand and practice the Racing Rules of Sailing (US-centric). Currently in early planning/research phase -- no code yet.

## Project Phases

The project follows a sequential phase plan (see README.md for full list):

1. Competitive analysis
2. Rules research and data collection
3. Pedagogy research for mobile learning
4. App Store pricing research
5. Go/no-go decision
6. Implementation approach research
7. Goal-setting (minimum value, MVP, final product)
8. Project plan creation

Development work begins only after phases 1-5 are complete and a go decision is made.
