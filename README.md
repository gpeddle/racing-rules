# README.md - racing-rules

This project is for creating a mobile app that helps people understand and practice the Racing Rules of Sailing (US-centric).

If monetization is viable, proceeds will be donated to youth or community sailing programs.

The project will consist of several phases, which must be completed in order:

1. research other applications that have a similar purpose, check for competitors who already satisfy this niche.
2. research and collect details on the racing rules of sailing
3. research anc collect details on mobile app pedegogy for this topic -what is most effective and engaging?
4. research and collect data na dguidance on pricing an app like this in the apple app store.
5. review all collected info and make a decision about whether to pursue development
6. research and collect info on implementation approaches
7. create a set of goals for the app's development timeline, including Min value, MPV, final product
8. create a project plan.
